Map demo

Rutas relevantes:
- `/map` muestra promociones cercanas con Leaflet.

Requerimientos:
- Asegurarse de tener `VITE_BACKEND_URL` en `.env` del frontend (ej: `VITE_BACKEND_URL=http://localhost:4000`).

Instalación (desde `frontend`):

```bash
npm install
npm run dev
```
# VecinAPP - Frontend

Stack: React + Vite + Tailwind + Leaflet

Instalación y ejecución:

```bash
# desde la carpeta raíz del proyecto
cd frontend
npm install
npm run dev
```

El dev server corre por defecto en `http://localhost:5173`.
